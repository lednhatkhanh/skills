# Go Skill Workspace

This repo contains a reusable modern Go skill and supporting references.

## What's Here

- `use-modern-go/SKILL.md`: main skill instructions
- `use-modern-go/references/`: detailed guidance (style, testing, concurrency, modern features, source basis)
- `docs/go-skill-prompt.md`: reusable prompt for generating/updating the skill

## Quick Update Flow

1. Open `docs/go-skill-prompt.md`.
2. Copy the rewritten prompt.
3. Use your preferred AI assistant to update `use-modern-go/SKILL.md` and related files in `use-modern-go/references/`.
4. Review changes and run your normal checks before commit.
